ANGULARJS第二天笔记

双向数据绑定

1. 单向数据绑定

经典的数据绑定，多是从 Model 到 View。AngularJS 也可以实现这样的单向数据绑定。这种单向数据绑定，只能从 Model 影响 View，而不能从 View 来影响 Model。

AngularJS 实现单向数据绑定的方式如下：

  • Template（客户端模板）

    • AngularJS 的表达式：{{expression}}
    • AngularJS 的指令：ng-bind指令等

  • Model（模型）

    • AngularJS 的控制器（Controller）定义模型中的数据
<div ng-app="myApp" ng-controller="myCtrl">
    名字: <input value="{{name}}">
</div>

<script>
    var app = angular.module('myApp', []);
    app.controller('myCtrl', function($scope) {
        $scope.name = "John Doe";
    });
</script>

2. 双向数据绑定

AngularJS 实现双向数据绑定，就是在从 Model 到 View 的基础上，实现从 View 到 Model。

AngularJS 实现双向数据绑定的方式如下：

  • Template（客户端模板）
    • 通过使用 AngularJS 的指令 ng-model 将 客户端模板与 Model 进行绑定。

值得注意的是：只有表单控制的值可以修改（View 中数据的变化，自动更新到 Model）。

  • Model（模型）
    • 通过 $scope 对象的 $watch() 方法进行监听，监听 View 中的数据变化。

$watch(watchExpression, listener, [objectEquality])方法：

参数 类型 描述
watchExpression String 表达式的值
listener function(newVal, oldVal, scope) 当 watchExpression 值变化时的回调函数
objectEquality boolean 默认值为 false 
<div class="container" ng-controller="c1">
    <input ng-model="userName">
    <p>只要input的value改变，Model数据userName会自动随之改变</p>
    <p>{{userName}}</p>
</div>
<script>
    angular.module('myModule1', ['ng']).
        controller('c1', function ($scope) {
            //监视一个Model数据的改变
            $scope.$watch('userName',function(){
                console.log('Model数据userName的值改变了：'+$scope.userName);
            });
        });
</script>

过滤器和函数

1. 过滤器

AngularJS 的过滤器是通过一个管道字符（|）和一个过滤器添加到表达式或指令中。

AngularJS 过滤器的语法如下：

  • HTML 模板中：
{{ expression | filter : fractionSize}}
  • JavaScript 代码中：
$filter(filter)(...)

1）number过滤器

number过滤器表示格式化数字为一个字符串。语法如下：

  • HTML 模板中：
{{ number_expression | number : fractionSize}}
  • JavaScript 代码中：
$filter('number')(number, fractionSize)
参数 类型 描述
number Number或String 要格式化的数字
fractionSize Number或String 小数部分的有效位数 
<script>
  angular.module('numberFilterExample', [])
    .controller('ExampleController', ['$scope', function($scope) {
      $scope.val = 1234.56789;
    }]);
</script>
<div ng-controller="ExampleController">
  <label>Enter number: <input ng-model='val'></label><br>
  Default formatting: <span id='number-default'>{{val | number}}</span><br>
  No fractions: <span>{{val | number:0}}</span><br>
  Negative number: <span>{{-val | number:4}}</span>
</div>

2）currency过滤器

currency过滤器表示格式化数字为指定的货币形式。语法如下：

  • HTML 模板中：
{{ currency_expression | currency : symbol : fractionSize}}
  • JavaScript 代码中：
$filter('currency')(amount, symbol, fractionSize)
参数 类型 描述
amount Number 向过滤器输入的值
symbol String 用于显示的货币符号或标示符
fractionSize Number 小数部分的有效位数 
<script>
  angular.module('currencyExample', [])
    .controller('ExampleController', ['$scope', function($scope) {
      $scope.amount = 1234.56;
    }]);
</script>
<div ng-controller="ExampleController">
  <input type="number" ng-model="amount" aria-label="amount"> <br>
  default currency symbol ($): <span id="currency-default">{{amount | currency}}</span><br>
  custom currency identifier (USD$): <span id="currency-custom">{{amount | currency:"USD$"}}</span>
  no fractions (0): <span id="currency-no-fractions">{{amount | currency:"USD$":0}}</span>
</div>

3）date过滤器

date过滤器表示格式化日期为指定格式。语法如下：

  • HTML 模板中：
{{ date_expression | date : format : timezone}}
  • JavaScript 代码中：
$filter('date')(date, format, timezone)
参数 类型 描述
date Date、Number或String 需要格式化的日期
format String 格式化规则。默认为 mediumDate
timezone String 格式化时的时区 
<span ng-non-bindable>{{1288323623006 | date:'medium'}}</span>:
    <span>{{1288323623006 | date:'medium'}}</span><br>
<span ng-non-bindable>{{1288323623006 | date:'yyyy-MM-dd HH:mm:ss Z'}}</span>:
   <span>{{1288323623006 | date:'yyyy-MM-dd HH:mm:ss Z'}}</span><br>
<span ng-non-bindable>{{1288323623006 | date:'MM/dd/yyyy @ h:mma'}}</span>:
   <span>{{'1288323623006' | date:'MM/dd/yyyy @ h:mma'}}</span><br>
<span ng-non-bindable>{{1288323623006 | date:"MM/dd/yyyy 'at' h:mma"}}</span>:
   <span>{{'1288323623006' | date:"MM/dd/yyyy 'at' h:mma"}}</span><br>

4）upperCase过滤器

upperCase过滤器表示格式化字符串为全大写。语法如下：

  • HTML 模板中：
{{ uppercase_expression | uppercase}}
  • JavaScript 代码中：
$filter('uppercase')()
<div ng-app="myApp" ng-controller="personCtrl">

<p>The name is {{ lastName | uppercase }}</p>

</div>

<script>
angular.module('myApp', []).controller('personCtrl', function($scope) {
    $scope.firstName = "John",
    $scope.lastName = "Doe"
});
</script>

5）lowerCase过滤器

lowerCase过滤器表示格式化字符串为全小写。语法如下：

  • HTML 模板中：
{{ lowercase_expression | lowercase}}
  • JavaScript 代码中：
$filter('lowercase')()
<div ng-app="myApp" ng-controller="personCtrl">

<p>The name is {{ lastName | lowercase }}</p>

</div>

<script>
angular.module('myApp', []).controller('personCtrl', function($scope) {
    $scope.firstName = "John",
    $scope.lastName = "Doe"
});
</script>

6）orderBy过滤器

orderBy过滤器表示对数组中的记录进行排序。语法如下：

  • HTML 模板中：
{{ orderBy_expression | orderBy : expression : reverse : comparator}}
  • JavaScript 代码中：
$filter('orderBy')(collection, expression, reverse, comparator)
参数 类型 描述
collection Array 需要排序的数组
expression String 设置排序的字段
reverse Boolean 如果为 true，则按照倒序排序
comparator Function 用于自定义排序规则的回调函数。如果省略，则使用内置函数。 
<div ng-controller="ExampleController">
  <table class="friends">
    <tr>
      <th>Name</th>
      <th>Phone Number</th>
      <th>Age</th>
    </tr>
    <tr ng-repeat="friend in friends | orderBy:'age'">
      <td>{{friend.name}}</td>
      <td>{{friend.phone}}</td>
      <td>{{friend.age}}</td>
    </tr>
  </table>
</div>
<script>
    angular.module('orderByExample1', [])
    .controller('ExampleController', ['$scope', function($scope) {
      $scope.friends = [
        {name: 'John',   phone: '555-1212',  age: 10},
        {name: 'Mary',   phone: '555-9876',  age: 19},
        {name: 'Mike',   phone: '555-4321',  age: 21},
        {name: 'Adam',   phone: '555-5678',  age: 35},
        {name: 'Julie',  phone: '555-8765',  age: 29}
      ];
    }]);
</script>

2. 函数

1）forEach

angular.forEach()函数用于遍历对象或数组。语法如下：

angular.forEach(obj, iterator, [context])
参数 类型 描述
obj Array或Object 需要遍历的数组或对象
iterator Function 遍历函数
context Object 遍历函数的上下文对象 
var values = [
    {name: 'misko', gender: 'Male'},
    {name: 'mary', gender: 'Female'}
];
angular.forEach(values, function(value, key) {
    console.log(key + ': ' + value.name + "," + value.gender);
});

2）lowerCase和upperCase

angular.lowercase()函数用于将字符串设置为全小写，angular.uppercase()函数用于将字符串设置为全大写。

var values = [
    {name: 'misko', gender: 'Male'},
    {name: 'mary', gender: 'Female'}
];
angular.forEach(values, function(value, key) {
    console.log(key + ': ' + angular.uppercase(value.name) + "," + angular.lowercase(value.gender));
});

3）toJson和fromJson

angular.toJson()函数用于序列化 JSON 格式字符串，angular.fromJson()函数用于反序列化 JSON 格式字符串。

console.log(angular.toJson({name: "misko", gender: "Male"}));

var obj = angular.fromJson('{"name":"mary"}');
console.log(obj.name);

服务

1. 控制器和作用范围

1）$scope

$scope叫做作用域对象，每个控制器实例都拥有一个$scope对象。$scope对象是 View（视图） 和 Controller（控制器）之间的纽带。

$scope对象是 AngularJS 创建控制器实例时自动创建的。

<div ng-app="myApp" ng-controller="myCtrl">

<h1>{{carname}}</h1>

</div>

<script>
    var app = angular.module('myApp', []);
    
    app.controller('myCtrl', function($scope) {
        $scope.carname = "Volvo";
    });
</script>

值得注意的是：通过$scope对象来控制 View（视图）对应的表达式的值。

2）$rootScope

$rootScope叫做根作用域对象，用于在各个控制器之间共享模型数据。$rootScope对象作用域整个 AngularJS 应用中。

<div ng-controller="c1" class="container">
    {{name}},{{company}},{{age}}
</div>
<div ng-controller="c2" class="container">
    {{name}},{{company}},{{age}}
    <div ng-controller="c3" class="container">
        {{name}},{{company}},{{age}}
    </div>
</div>
<script>
    angular.module('myModule1', ['ng'])
        .controller('c1', function ($scope,$rootScope) {
            $rootScope.company = "TARENA";
            $scope.name = "mary";
        }).controller('c2', function ($scope) {
            $scope.name = "jerry";
            $scope.age = 39;
        }).controller('c3', function ($scope) {
            $scope.name = "tom";
            $scope.company = "ALI";
        })
</script>

2. 定时器

1）$interval

AngularJS 提供 $interval 对应 window.setInterval() 函数。语法如下：

$interval(fn, delay, [count], [invokeApply], [Pass])
参数 类型 描述
fn Function 被重复调用的函数
delay Number 设置每次调用函数间隔的毫秒数
count Number 设置重复的次数。如果没有设置，或设置为 0，则表示无限制重复
invokeApply Boolean 如果设置为 false，则跳过模型的脏检查。默认为 true
Pass * 为执行的函数添加参数 
<div ng-app="myApp" ng-controller="myCtrl"> 
    <p>现在时间是:</p>
    <h1>{{theTime}}</h1>
</div>

<p>$interval 访问在指定的周期(以毫秒计)来调用函数或计算表达式。</p>

<script>
    var app = angular.module('myApp', []);
    app.controller('myCtrl', function($scope, $interval) {
        $scope.theTime = new Date().toLocaleTimeString();
        $interval(function () {
            $scope.theTime = new Date().toLocaleTimeString();
        }, 1000);
    });
</script>

2）$timeout

AngularJS 提供 $timeout 对应 window.setTimeout() 函数。语法如下：

$timeout([fn], [delay], [invokeApply], [Pass])
参数 类型 描述
fn Function 被重复调用的函数
delay Number 设置每次调用函数间隔的毫秒数
invokeApply Boolean 如果设置为 false，则跳过模型的脏检查。默认为 true
Pass * 为执行的函数添加参数 
<div ng-app="myApp" ng-controller="myCtrl"> 
    <p>两秒后显示信息:</p>
    <h1>{{myHeader}}</h1>
</div>

<p>$timeout 访问在规定的毫秒数后执行指定函数。</p>

<script>
    var app = angular.module('myApp', []);
    app.controller('myCtrl', function($scope, $timeout) {
      $scope.myHeader = "Hello World!";
      $timeout(function () {
          $scope.myHeader = "How are you today?";
      }, 2000);
    });
</script>

3）$scope.$digest

$scope.$digest()方法用于手工触发 $scope.$watch() 队列中的所有监听的函数。

需要注意的是：通常不需要直接调用$scope.$digest()方法，而是通过调用$scope.$apply()方法进行进一步的封装。

$scope.name = 'misko';
$scope.counter = 0;

expect($scope.counter).toEqual(0);
$scope.$watch('name', function(newValue, oldValue) {
  $scope.counter = scope.counter + 1;
});
expect($scope.counter).toEqual(0);

$scope.$digest();
// the listener is always called during the first $digest loop after it was registered
expect($scope.counter).toEqual(1);

$scope.$digest();
// but now it will not be called unless the value changes
expect($scope.counter).toEqual(1);

$scope.name = 'adam';
$scope.$digest();
expect($scope.counter).toEqual(2);

4）$scope.$apply

$scope.$apply()方法用于执行一个在 AngularJS 框架上下文环境外的 AngularJS 表达式。语法如下：

$scope.$apply();//不推荐使用

$scope.$apply(function(){//推荐使用
       //保证任何对模型数据的修改，
       //都可以被监听函数处理
});

3. Ajax调用

$http服务

$http服务是 AngularJS 的一个核心服务，该服务用于通过 XMLHttpRequest 对象或 JSONP 与远程 HTTP 服务进行通信。

$http服务的基本使用
$http({
  method: 'GET',
  url: '/someUrl'
}).then(function successCallback(response) {
    // this callback will be called asynchronously
    // when the response is available
  }, function errorCallback(response) {
    // called asynchronously if an error occurs
    // or server returns response with an error status.
  });

response对象具有以下属性：

  • data - String或Object类型，响应数据。
  • status - Number类型，HTTP 的响应状态码。
  • statusText - String类型，HTTP 的响应状态信息。
$http服务的快捷方法
  • $http.get
  • $http.post
  • $http.jsonp
<div ng-app="myApp" ng-controller="customersCtrl"> 
    <ul>
      <li ng-repeat="x in names">
        {{ x.Name + ', ' + x.Country }}
      </li>
    </ul>
</div>

<script>
    var app = angular.module('myApp', []);
    app.controller('customersCtrl', function($scope, $http) {
      $http.get("Customers.json")
      .success(function (response) {$scope.names = response.records;});
    });
</script>
