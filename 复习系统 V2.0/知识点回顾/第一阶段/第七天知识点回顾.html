逻辑运算符
逻辑运算符
1.与运算（&&）：可以对符号两侧的值进行与运算。只要一个为false就返回false。
2.或运算（||）.可以对符号两侧的值进行或运算，只要一个为true就返回true。
3.逻辑运算
（1）.（&&）运算找false， 如果第一个值为false，则返回第一个。如果第一个值为true，则返回第二个
（2）.（||）运算找true，如果第一个值为true，则返回第一个。如果第一个值为false，则返回第二个
关系运算符
关系运算符
关系运算符：可以比较两个值之间的大小关系
变现形式有：
(1). >= 比较符号左侧的值是否大于等于右边的值，关系成立，返回true，否则false
(2). >比较符号左侧的值是否大于右边的值，关系成立，返回true，否则false
(3). <= 比较符号左侧的值是否小于等于右边的值，关系成立，返回true，否则false
(4). >比较符号左侧的值是否小于右边的值，关系成立，返回true，否则false
赋值运算符
赋值运算符
1.将等号右边的值，赋值给等号左边的变量
赋值的形式：
(1). -=等同于a=a-1
(2). +=等同于a=a+1
(3). *=等同于a=a*1
(3). \=等同于a=a\1
(3). %=等同于a=a%1
相等运算符
相等运算符
(1). == 用来判断符号两边的值是否相等，如果成立，返回true，否则false
(2). === 用来判断符号两边的值和类型是否都相同，如成立，返回true，否则false
三元运算符
三元运算符
(1). 三元运算符需要三个操作数
(2). 语法：表达式1？语句1：语句2（可以为值，也可是表达式）
(3). 三元运算符当运算成立时执行-语句1，当运行不成立时执行-语句2
运算符优先级
运算符优先级
(1). 优先级越高的运算符，优先计算
(2). 括号的优先级较高，可应用提升代码的可读性。
(3). * \ 高于 + -
(4). && 高于 ||
流程控制语句(if)
流程控制语句(if)
结构：
键盘输入函数：prompt（）；
(1). if (条件成立) { 语句。。。 }
(1). if (条件成立) { 语句。。。 } else { 条件不成立执行语句。。。 }
(1). if (条件成立) { 语句。。。 } else if (条件成立) { 语句。。。 } else { 条件不成立执行语句。。。 }
if的判断条件返回值，成立时为true，否则为false
