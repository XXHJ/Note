对象
对象属于引用类型(Object)，对象是一种复合类型的数据，对象就像是一个容器，可以装任何类型的属性
对象分类：
(1).内建对象：内建对象指由ES标准定义的对象，任何对ES标准的实现都可以使用这些对象，例：Math String Number Boolean RegExp
(2).宿主对象：宿主对象指由JS运行环境为我们提供的对象。例：BOM对象、DOM对象
(3).自定义对象：自定义创建的对象
创建对象：
  语法：对象.属性名
(1).调用构造函数创建对象：var obj = new Object()
(2).对象字面量：var obj = { }
  新增对象属性：obj.name = "孙悟空";
  修改对象属性值：obj.name = "沙和尚";
  删除对象属性：delete obj.name
  查询对象属性：obj.name
注意:对象的属性值可以为任意的数据类型
基本和引用数据类型
基本和引用数据类型
(1).基本类型数据的变量名和值都存储在栈中
(2).引用类型数据，其对象名和内存地址存储在栈中，对象开辟的对象空间存储在堆中，通过栈中的内存地址把对象引用起来
对象属性
对象属性
添加对象方式：
(1).方式一：对象.属性名 = 属性值 ，例子：obj.name = "孙悟空";
(2).方式二：对象["属性名"] = 属性值，例子：obj["name"] = "孙悟空";
(3).对象的命名可以任意命名
(4).属性的属性值可以是任意数据类型，也可以是一个对象、一个函数
对象字面量 obj={}
对象字面量 obj={}
语法：var 对象名 = {属性：属性值，属性：属性值，属性：属性值。。。}
属性值可以为任意数据类型，以及对象和函数
垃圾回收(GC)
垃圾回收(GC)
(1).当一个对象没有任何变量对其进行引用时，就会变成垃圾对象
(2).浏览器都有自动清理垃圾的机制。
(3).当对象在栈中的内存地址为空时，就会产生一个垃圾对象。例子:obj = null;
(3).释放对象方式：obj = null;
属性的枚举(for... in...)
属性的枚举 (for... in...)
(1).使用 for...in 语句对一个对象中的属性进行枚举(输出一个对象中的所用属性)
(2).语法：for (变量 in 对象) { 输出对象中的所用属性(对象[变量]) }
(2).举例：for(var key in obj){document.write(key+":"+obj[key]+"<换行符br>");}
函数(function)
函数(function)
(1).函数也是一个对象，也具体Object的所有功能
(2).不同于对象的是，它可以封装一些执行代码，并且在需要的时候执行这些代码
创建函数对象：
(1).创建函数方式一(命名函数)：function ([形参1，形参2...]) { 语句... }
(2).创建函数方式二(匿名函数)：var obj = function ([形参1，形参2...]) { 语句... }
(3).函数的调用：函数名([实参1,实参2...])
(4).在函数调用时，形参和实参的个数可以不同，但都是从左到右进行匹配的
(5).在函数调用时，传递的实参可以是任意类型的数据
(6).在函数调用时，默认都会有一个返回值(Undefined)。指定的返回值也可以是任意类型
(6).在函数调用时，语句执行到return时，立即结束函数。如果return后面含有代码讲不会执行
