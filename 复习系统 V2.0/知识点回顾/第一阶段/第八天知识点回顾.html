switch分支语句
语法结构：
switch(条件表达式){ case 值1 ： 表达式1 break；case 值2 ： 表达式2 break；default ： 表达式 break；}
特点：
语句中寻找全等值，寻找到全等值后，会执行相应的表达式，如果没有寻找到，则执行default语句后的表达式
案例：
(1).通过输入一个（1-7）的数字来判断星期几？
解决方案：
num = +prompt("输入星期几"); switch(num){ case 1 : console.log("壹"); break; case 2 : console.log("贰"); break; case 3 : console.log("叁"); break; case 4 : console.log("肆"); break; default : console.log("非法字符"); break;]] }
while循环
while循环
语法结构：
while（条件表达式）{ 语句。。。 }
特点：
(1).当条件表达式的值为true时，执行循环体的语句
(2).当条件表达式的值为false时，不会执行循环体内的语句
案例：
(1).假设：投资的年利率是5%，问：1000元投资多少年能变成5000?
解决方案：
//创建一个变量，来保存钱数 var money = 1000; //定义一个计数器，来记录循环执行的次数 var count = 0; //创建一个while循环 while(money < 5000){ money *= 1.05; //对计数器进行自增 count++; } console.log("money = "+money); console.log("count = "+count);
do-while循环
do-while循环
语法结构：
do { 循环语句。。。 } while (循环条件)
特点：
(1).当条件表达式的值为true时，执行循环体的语句
(1).当条件不成立时，do-while循环会至少执行一次
for循环
for循环
语法结构：
for (初始化表达式1，条件表达式2，更新表达式3) { 循环体语句4。。。}
特点：
(1).执行顺序：1——2——4——3
(2).执行一次后会重复执行2——4——3，直到条件2不成立（为false）

(1).打印9*9乘法口诀表
