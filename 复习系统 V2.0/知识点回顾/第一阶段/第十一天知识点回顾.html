数组对象
数组对象(Array)
特点：
  1.数组也是一个对象
  2.数组中可以保存任意类型的元素
  3.数组是通过索引(index)来操作内部存放的元素的
  4.数组存储的效率要比对象要高，在开发中经常会使用数组来存储数据
创建数组：
  方式一(new Array)：
     var arr = new Array();
  方式二(数组字面量)：
     var arr = [];

数组方法
数组方法
添加数组元素：
  1.push()方法：向数组最后的一个元素添加一个或多个元素，并返回数组的长度
    用法：arr.push("元素一","元素二"...)
  2.unshift()方法：向数组第一个元素前添加一个或多个元素，并返回数组的长度
    用法：arr.unshift("元素一","元素二"...)
  3.pop()方法：删除数组最后一个元素，并返回删除的元素
    用法：arr.pop();
  4.shift()方法：删除数组第一个元素。并返回删除的元素
    用法：arr.shift()
  5.slice(参数一，参数二)方法：返回选定的元素，带有两个参数：参数一指要返回选定值的开始索引，参数二，选定的结束参数(但不包含结束的索引)，不会改变原数组。
    用法：arr.slice(开始索引，结束索引)
  6.splice(参数一，参数二，参数三)方法：删除指定的是元素，并添加新元素。参数一，开始删除的索引；参数二，删除的个数；参数三，新增的元素。会改变原数组
    用法：arr.splice(开始删除的索引，删除的个数，新增的元素)
  7.reverse()方法：讲数组元素反转，会改变原数组
    用法：arr.reverse();
  8.sort()方法：对数组进行排序，默认的是Uncode编码的排序。
    用法(对数字进行排序)：arr.sort(function (a,b) { return a-b;});
  9.concat()方法：多个数组拼串在一起
    用法：arr.concat(arr1,arr2...)
  10.join()方法：将数组转换成字符串，并且可以指定字符将数组拼凑起来成一个字符串
    用法：arr.join("拼接的字符串")

验证数组练习(Array)
练习一：过滤掉数组中出现重复的数组？
方案一：
  var arr = [1,2,3,1,1,1,4,2,3,5];
  for(var i=0 ; i<=arr.length-1; i++){ //获取数组中的数字
    for(var j=i+1 ; j
      if(arr[i] == arr[j]){ //比较两个数字是否相等
        arr.splice(j,1); //进入判断，证明两个数字重复，则删除arr[j]
        j--; //需要在此位置在比较一次
      }
    }
  }
方案二：
  var arr = [1,2,3,1,1,1,5,6,7,7,8,8,3,3,3,4,2,3,5]; //创建数组
  var arr1 = arr.sort(function(a,b) {return a-b;}); //对数组进行排序
  for (var i=0; i
    if (arr1[i]==arr1[i+1]) { //进行比较，判断是否有重复的
      arr1.splice(i+1,1); //遍历到了重复的就删除
      i--;
    }
  }
函数知识的补充
函数知识的补充(Array)
1.在函数调用时，浏览器还提供了一个隐含的参数(arguments)
2.arguments是一个伪类数组对象，存储函数执行时的实参
3.arguments中有一个属性 callee，代表的是当前正在调用的函数对象
  用法：arguments.callee==正在调用的对象
4.isArray()检验一个对象是否是数组
  用法：Array.isArray(对象)
5.函数对象的方法：
  (1).call()和apply()方法
  当调用函数对象的call和apply时都会导致函数立即执行，并且可以指定调用函数
  用法：对象.方法.call(指定的对象，参数。。。)
  用法：对象.方法.apply(指定的对象，参数要打包成数组)
  通过call和apply调用时，第一个参数会成为this
call和apply的区别：
call的参数，需要一个一个列出来；apply的参数，需要封装为一个数组传递
