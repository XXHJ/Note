BOM对象
BOM对象
BOM浏览器对象模型：提供一组对象，可以通过这些对象来操作浏览器
(1).window对象：代表浏览器窗口
(2).navigator对象：浏览器版本信息
(3).history对象：浏览器的访问历史记录
(4).location对象：浏览器地址栏，跳转地址
(5).screen对象：代表浏览器视口信息
navigator对象
navigator对象
获取浏览器版本信息：
  (1).正常浏览器通过navigator.userAgent来获取浏览器的版本信息
  (2).IE11浏览器痛过window.ActiveXObject方法来进行判断
  (3).IE11以下可以通过navigator.userAgent来判断其是否具有MSIE标识
浏览器版本判断案例：
  if(/firefox/i.test(ua)){
    alert("你是火狐~~~");
  }else if(/chrome/i.test(ua)){
    alert("你是chrome浏览器~~~");
  }else if(/msie/i.test(ua)){
    alert("你是IE");
  }else if("ActiveXObject" in window){
    alert("你是IE11");
  }
注意：需要使用到正则表达式来尽心判断，其中火狐是通过/firefox/i.test(ua)，谷歌数通过/chrome/i.test(ua)，IE是通过"ActiveXObject" in window来判断
在IE11情况下!!window.ActiveXObject的值是为false
history对象
history对象
history对象：标识对象用户访问的历史记录，从打开浏览器开始记录，到浏览器关闭就结束。
(1).length属性：表示记录访问历史记录的个数
(2).back()方法：会退到上一步
(3).forward()方法：跳转到下一步
(4).go()方法：跳转带指定的页面，其中go()方法中传递的是整数值，整数值为正的时候向前，负值向后
location对象
location对象
location对象，控制浏览器地址栏的信息
location属性：
  (1).href属性：获取完整的URL路径
  (2).port属性：设置或获取端口号
  (3).protocol属性：设置或返回URL所使用的协议
  (4).search属性：设置或返回从问号 (?) 开始的 URL（查询部分）
  (5).pathname属性：设置或返回当前 URL 的路径部分
location方法：
  (1).assign():跳转到指定的路径，与直接使用location=“路径”效果相同，能够产生对象用户的历史记录
  (2).reload():页面重新加载，可以利用此功能实现页面刷新；例：location.reload(true);如果参数中不带true则不能进行强制刷新
  (3).replace():用新的地址替换当前地址，此方法不会产生历史访问记录
window对象(定时器)
window对象(定时器)
定时器方法(一直执行)：setInterval(回调函数-function，定时执行时间-毫秒)
取消定时器方法：clearInterval(需要关闭定时器的标识符)
定时调用指每间隔一段时间就调用指定的函数一次，并会返回一个Number类型的值
例子：var timer = setInterval(function() {},1000);
延时方法(只执行一次)：setTimeout(回调函数-function，延时执行时间-毫秒)
取消延时的方法：clearTimeout(需要关闭延时函数的标识符)
延时调用指函数不马上执行，而是过一段时间以后再执行，并会返回一个Number类型的值，与定时器的使用方法相同
其中定时函数和延时函数可以相互之间实现对方的功能
