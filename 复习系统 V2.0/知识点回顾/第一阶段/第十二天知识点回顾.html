日期时间对象
日期时间对象(Array)
创建Date对象：
  var dt = new Date();
调用时间对象：
  getDate()可以获取当前日期对象是几号: dt.getDate()
  getDay()可以获取当前日期对象时周几(取值范围0-6): dt.getDay()
  getMonth()可以获取当前日期对象的月份(取值范围0-11)： dt.Month()
  getFullYear()获取当前日期对象的年份: dt.getFullYear()
  getTime()获取当前日期对象的时间戳.时间戳指从格林威治标准时间1970年1月1日0时0分0秒 到 现在所花费的 毫秒数,在计算机的底层，所有的时间都是以时间戳的形式保存的
  用法：dt.getTime()
  Date.now()可以获取代码执行时的时间戳,这个方法不用创建对象：Date.now()
  
  
Math内置对象
Math内置对象
Math中的量：
  1.Math.PI:求圆周率
  2.Math.e:求自然对数的底数
Math中方法：
  1.Math.abs(参数):求一个数的绝对值
  2.Math.sqrt(参数)：求一个数的开平方根
  3.Math.pow(参数一，参数二):求一个数的幂，参数一表示底数，参数二表示幂的次数；等同pow(3,4)=3*3*3*3
  4.Math.ceil(参数):对参数向上取整，例1.1=2
  5.Math.floor(参数):对参数向下取整，例1.9=1
  6.Math.round(参数):对参数进行四舍五入，例 1.1=1；1.5=2
  7.Math.random():求随机数，随机数的范围是0~1，
    例子：求一个范围(x-y)的范围的随机数？
    var ran = Math.round(Math.random()*(y-x)+x);
  8.Math.max(参数):求参数的最大值
  9.Math.min(参数):求参数的最小值
ToString对象
ToString对象
特点：浏览器在解析时，保存字符串实际上是以字符数组的形式保存的
  1.可以通过length来获取字符串的长度
  2.charAt(索引):根据指定的索引来获取对应的字符
  3.charCodeAt(索引):根据指定的索引来获取对应的Unicode编码
  4.fromcharCode(字符编码):根据指定的字符编码来获取对应的字符
  5.concat(字符串一，字符串二....):用于将一个或多个字符串拼接在一起
  6.indexOf(参数一，参数二):从前往后检索指定的字符串在当前字符串的位置。参数一：检索的字符串；参数二，开始查找位置。默认会返回第一个出现要检索的字符串的位置，否则返回-1
  7.lastIndexOf(参数一，参数二):从后往前检索指定的字符串在当前字符串中的位置，参数一：检索的字符串；参数二：开始查找的位置。默认会返回第一个出现要检索的字符串的位置，否则返回-1
  8.slice(参数一，参数二):截取指定位置的长度的字符串。参数一：开始的位置；参数二：结束的位置（不包含结束的索引），当参数二为负数的时候就从后往前截取
  9.subString(参数一，参数二):截取指定位置的长度的字符串。参数一：开始截取的位置，参数二：结束为的位置（不包含结束的索引），参数二为负数的时候默认为0，参数一大于参数二，会调整参数一和参数二位置
  10.substr(参数一，参数二):截取字符串。参数一：开始截取的位置；参数二：截取的长度
  11.split(参数):将一个字符串拆分成数组。根据指定参数来拆分字符串
  12.toLowerCase(参数):将字符串转换成小写
  13.toUpperCase(参数):将字符串转换成大写
正则表达式
正则表达式
特点：
  1.用来定义一些规则，计算机可以根据这些规则来判断一个字符型是否符合规则，或者从一个字符串中将符合规则的内容提取出来
创建一个正则表达式：
var reg = new RegExp("正则表达式"，"匹配模式(i:忽略大小写，g：全局匹配，m：多行匹配)");
创建正则表达式字面量：var reg = /正则表达式/匹配模式;
test(需要验证的字符串)方法：检查一个字符串是否符合正则表达式的要求。符合返回true，否则false
正则表达式设计：
(1)量词:[]括号中的内容表示或
  [0-9]:任意数字  [a-z]:任意小写字母  [A-Z]:任意大写字母  [A-z]:任意字母
  [^ ]:表示除了括号中的内容  
  n{x}:表示n正好出现x次  n{x,y}:表示n出现x-y次  n{x，}:表示n至少出现x次
  n+:表示至少出现1次，n+=n{1，}  n*:表示出现0次或多次，n*=n{0，}  n？；表示出现0次或1次，n？=n{0,1}
  ^:表示开头  $:表示结尾
  .:在正则表达式表示任意字符  可以使用 \ 作为转义字符
量词简写：\w:代表任意字母数字下划线；\W:则是除了字母数字下划线
  \d:代表任意数字；\D:则是除了数字
  \s:代表空格；\S则是除了空格
  \b:代表单词边界；\S则是除了单词边界
正则表达式支持String对象的方法：
  1.split(正则表达式):将字符串拆分成数组，自动全局查找
  2.search(正则表达式):检索指定的项，不管匹配模式，只会检索遇到的第一个
  3.macth(正则表达式):将匹配到的内容提取出来，如不设置匹配模式，只会提取第一个
  4.replace(正则表达式,"替换的内容"):将匹配到的内容替换，如不指定匹配模式，只会更新第一个
正则表达式案例：
案例一：验证QQ？
  正则表达式：/^[1-9]{1}[0-9]{4,10}$/
案例二：验证邮箱？
  正则表达式一：/^(\w[_|\_|\.]?){1,16}@(\w[_|\_|\.]?){2,5}\.+([a-z|A-Z]){2,5}$/;
  正则表达式二：/^(\w[_|\_|\.]?){1,16}@[qq.com|163.com|guge.com]$/
  正则表达式三：/^([a-z|A-Z|0-9][_|\_|\.]?){1,13}@([a-z|A-Z|0-9][_|\_|\.]?){2,3}\.+([a-z|A-Z]){2,3}$/
