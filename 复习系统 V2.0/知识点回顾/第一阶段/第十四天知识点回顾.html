DOM其他方法
DOM其他方法
(1).获取body和head标签：var body = document.body;或var head = document.head;
(2).获取根标签（html）方法：var html = document.documentElement;
(3).获取页面中所有的元素对象:var all = document.getElementsByTagName("*");或 var all = document.all(建议使用)
(4).获取类名的方法: var className = document.getElementsByClassName("类选择器名");或 var className = document.querySelector("id和类选择器都可以，建议使用"),只会返回第一个选择器
(5).获取所用相同的类选择器:var box1 = document.querySelectorAll(".box1");会返回所有的选择器
(6).返回标题内容: var title = document.title;
DOM增删改
DOM增删改
(1).appendChild("新增内容"):将指定的内容插入到指定的节点上（注意需要找到指定插入元素的父节点：元素.parentNode.appendChild("插入的内容");）
(2).insertBefore("新元素"，"旧元素"):在指定的子节点前面插入新的子节点。（注意在使用前要找到需要其插入目标节点的父节点：元素.parentNode.insertBefore("新元素"，"旧元素");）
(3).replaceChild("新元素"，"旧元素"):替换子节点。（注意在使用前要找到需要其插入目标节点的父节点：元素.parentNode.replaceChild("新元素"，"旧元素");）
(4).removeChild("需要删除的节点"):删除子节点。注意在使用前要找到需要其插入目标节点的父节点：元素.parentNode.removeChild("需要删除的节点");）
(5).createElement("元素标签"):创建元素节点。用法（创建一个li标签）： var li = document.createElement("li");
(6).createTextNode("文本内容"):创建文本节点。用法：var liText = document.createTextNode("文本内容");
(7).innerHTML的巧用：可以使用innerHTML来进行HTML代码的创建和修改工作。
