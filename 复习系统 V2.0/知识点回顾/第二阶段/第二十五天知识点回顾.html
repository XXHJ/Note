背景渐变
背景渐变
1.线性渐变 linear-gradient()
 设置渐变的方向 to left/top/bottom/right 或者是旋转角度deg,原点在下边,顺时针旋转
 渐变的结构
  .box {
   background-image: linear-gradient(
    to right, red 50px,yellow 50px, yellow 100px, blue
   );
  }
 重复渐变的结构
  .box {
   background-image:repeating-linear-gradient(
    rgba(255, 0, 0, .3) 0
    , rgba(255, 0, 0, .3) 20px
    , rgba(255, 255, 0, .3) 20px
    , rgba(255, 255, 0, .3) 40px
    );
  }
2.径向渐变 radial-gradient()
 设置径向渐变的类型：
  圆形： circle - 圆形的渐变效果
  椭圆: ellipse - 椭圆的渐变效果（默认值）
  注意：书写方式 ： 渐变形状 渐变半径（椭圆有两个X/Y，为数值和百分比） at 渐变中心点（以元素左上角开始计算，接收关键字和百分比以及数值）
   控制圆形渐变的半径 以及中心点 中心点的位置以元素的左上角计算
   椭圆的半径设置支持百分比和数值设置,圆的目前只支持半径为数值
圆形渐变结构：
.box {
  background-image:radial-gradient(
    circle 50px at center, red , blue
  );
}
椭圆渐变结构：
.box {
  background-image:radial-gradient(
    ellipse 100% 50% at center, red , blue
  );
}
重复径向渐变的结构：
.box {
  background-image:repeating-radial-gradient(
    circle 5px at center, red 0,red 10px,blue 10px,blue 20px
  );
}

页面布局
怪异模式下计算盒子宽度是包含边框到内容区域，设置使用怪异模式的方式：box-sizing：border-box;
如果是因为外边距导致的空间不足，只能修改元素的大小，可以使用计算函数（兼容性问题）
 使用方法：width:calc()
使用案例：
 width: calc(200px - 1px);
 width: calc(200px + 100px);
 width: calc(200px / 2);
右外边距失效;当父元素的宽度不足以撑开子元素的左右外边距的时候，会是右边的外边距失效
