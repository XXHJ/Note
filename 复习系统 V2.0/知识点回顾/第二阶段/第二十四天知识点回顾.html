动画-Animation
动画a:animation
定义动画使用关键字 @keyframes 关键帧动画名
  from {起始帧} ， to {结束帧} ，10% {整体动画时间的10%}
定义关键帧的示例：
  @keyframes anmi }
    from {
      需要过渡的关键帧，具体为相关变化的样式
    }
    10% {
      需要过渡的关键帧，具体为相关变化的样式
    }
    20%,50% { //具有相同变化的样式可以使用分组器来进行简写
      需要过渡的关键帧，具体为相关变化的样式
    }
    to {
      需要过渡的关键帧，具体为相关变化的样式
    }
1.动画相当于一个高级的过渡效果，不用触发因素也可以执行，在对应元素的样式里引用animation-name:关键帧名
2.动画执行的时间:animation-duration:3s
3.动画执行的延时时间:animation-delay:2s
4.规定动画的是否保留开始帧与结束帧的状态:animation-fill-mode:backwards(保留起始帧)/forwards(保留结束帧)/both(起始/结束都保留)
5.规定动画播放的次数:animation-iteration-count:n(循环的次数)/infinite(无限循环)
6.规定动画播放的顺序:animation-direction:reverse(反向播放)/alternate(交替反复播放)
  注意:播放顺序要与动画执行次数配合使用，至少播放的次数大于1次，才能看到效果
7.规定鼠标进入动画时,动画是否暂停:animation-play-stata:running(不暂停)/paused(暂停)
  注意:此属性不能纳入animation简写里面
8.动画执行的方式：animantion-timing-function:linear(匀速)，ease，ease-in，ease-out ease-in-out，cubic-bezier(n,n,n,n)贝塞尔曲线
animation简写：
animation：关键帧名 动画执行时间 动画延时时间 动画执行方式 动画播放次数 动画播放顺序

图片边框
图片边框
图片边框：border-image
1.引入图片边框的路径:boeder-image-source:url("图片路径")
2.图片边框的切割:border-image-slice:fill(默认填充)/
  border-image-slice: 52 26 26 26;
  可以通过上右下左的方向进行分割
3.图片边框的平铺:border-image-repeat:repaat(平铺)/round(铺满平铺)/stretched(拉伸)
4.图片边框的宽度:border-image-width:length(具体的像素值);设置图片边框的宽度不会影响元素相邻的兄弟元素
5.图片边框的向外扩张:border-image-outset:length(具体的像素值);不会影响相邻元素的布局

背景裁剪
背背景裁剪景
1.背景图片的中心设置：
 1.background-origin:border-box(基于元素边框左上角来定位)
 2.background-origin:padding-box(基于元素内边距左上角来定位)
 3.background-origin:content-box(基于内容区域左上角开始定位)
2.设置图片的开始位置：
 background-position:x方向 y方向;
3.规定背景绘制区域：
 background-cilp:border-box(背景被裁剪到边框盒)
 background-cilp:padding-box(背景被裁剪到内边距盒)
 background-cilp:content-box(背景被裁剪到内容区)
4.背景是否平铺：
 background-repeat:repeat-x(水平X平铺)/repeat-y(水平Y平铺)/no-repeat(不平铺)
5.设置背景图片的大小：
 background-size:length/length 通过像素值设置背景图片的大小
 background-size:contain 把图像扩展至最大尺寸，以使其宽度和高度完全适应内容区域。
6.文本裁剪：
 -webkit-background-clip:text
 -webkit-text-fill-color: transparent;
说明：实现背景在文本内容中显示，即文本的裁剪
