ANGULARJS第一天笔记

AngularJS概述

1. 什么是AngularJS

AngularJS 诞生于 2009 年，由 Misko Hevery 等人创建，后为 Google 所收购。是一款优秀的前端 JavaScript 框架，已经被用于 Google 的多款产品当中。

AngularJS 的官方文档是这样介绍它的：

完全使用 JavaScript 编写的客户端技术。同其他历史悠久的 Web 技术（HTML、CSS 和JavaScript）配合使用,使 Web 应用开发比以往更简单、更快捷。

AngularJS 主要用于构建单页面 Web 应用，考虑的是构建 CRUD（增加Create、查询Retrieve、更新Update、删除Delete）应用。

以下就是一个很简单的 AngularJS 的示例代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>AngularJS初体验</title>
    <link rel="stylesheet" href="css/bootstrap.css">
    <script src="js/angular.js"></script>
</head>
<body ng-app>
    <fieldset>
        <legend>动态控制样式</legend>
        <input type="text" ng-model="data.cls">
        <div class="{{data.cls}}">这是一个{{data.cls}}</div>
    </fieldset>
</body>
</html>

2. AngularJS的核心思想

1）客户端模板

创建一个 hello.html 页面，具体代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>客户端模板</title>
    <script src="js/angular.js"></script>
</head>
<body>
    <div ng-app="myApp" ng-controller="HelloController">
        <p>Hello, {{text}}</p>
    </div>
    <script>
        var app = angular.module("myApp",[]);
        app.controller("HelloController",function($scope){
            $scope.text = 'AngularJS';
        });
    </script>
</body>
</html>

上述代码运行的结果会在浏览器显示：

Hello, AngularJS

通过上述示例，会发现 AngularJS 与之前的一些方式的不同：

  • 并没有使用 DOM 操作页面中<p>元素中的文本内容。
  • controller.js 用于控制 HTML 页面中的内容。
  • HTML 页面中的 {{text}} 内容会跟随 controller.js 中 $scope.text 的值改变而改变。

AngularJS 将 HTML 元素与数据进行分离，HTML 页面演变成一个模板，根据数据的变化而动态显示效果。

2）Model View Controller（MVC）

MVC 的核心理念是：将管理数据的代码（model）、应用逻辑代码（controller），以及向用户展示数据的代码（view）清晰地分离开。

在 AngularJS 应用中，视图（view）就是 DOM，控制器（controller）就是 JavaScript 代码，而模型数据（model）则被存储在对象的属性中。

在1）客户端模板中的示例代码中：

  • <p>Hello, {{text}}</p>作为 AngularJS 中的视图。
  • HelloController函数作为 AngularJS 中的控制器。
  • $scope.text = 'AngularJS';作为 AngularJS 中的模型数据。

3）数据绑定

在 Ajax 实现的单页应用中，会把 HTML 元素和数据混合起来，刷新 DOM 中的局部内容，而不是刷新整个页面。

但上述效果需要通过 JavaScript 代码实现，如果不需要编写任何代码就可以实现呢？在 AngularJS 中提供了一种叫做数据绑定。通过以下示例代码了解一下：

<div ng-app="myApp" ng-controller="HelloController">
    <input type="text" ng-model="text">
    <p>Hello, {{text}}</p>
</div>

4）依赖注入

在1）客户端模板中的示例代码中，进行数据绑定的 $scope 对象会呗自动传递，不需要调用任何函数创建这个对象。

这种通过某个指定对象进行数据的传递（将模型数据传递给视图中显示），是通过 AngularJS 的依赖注入机制实现的。

5）指令

AngularJS 最强大的功能之一就是指令：

  • 在 HTML 页面中的双花括号用于实现数据绑定。
  • ng-controller用于指定每个控制器负责监听视图中的哪一部分。
  • ng-model用于把输入数据绑定到模型中的一部分属性上。

3. 如何使用 AngularJS

1）引入 AngularJS 文件

<script src="js/angular.js"></script>

2）使用 ng-app 指令创建模块

AngularJS 是模块化的，MVC 是基于某个具体模块的。ng-app指令表示创建模块，一般设置在 HTML 页面的根标签或指定范围的容器标签中。

<div ng-app="myApp"></div>

通过以下 JavaScript 代码创建一个新的模块。

var app = angular.module("myApp",[]);

创建、修改或检索一个 AngularJS 的模块的语法如下：

angular.module(name, [requires], [configFn]);
参数 类型 描述
name String 创建或检索的模块名称
requires Array 如果指定，则创建新的模块。如果未指定，则使用现有模块并进一步配置。
configFn Function 配置模块，类似于module.config()方法

值得注意的是：ng-app指令的值与angular.module()函数中的第一个参数值要保持一致。

3）使用 AngularJS 的表达式

<div ng-app="myApp" ng-controller="HelloController">
    <p>Hello, {{text}}</p>
</div>

通过上述示例代码可以知道，AngularJS 的表达式被编写在双花括号中：{{ expression }}。

4）使用 AngularJS 的控制器或模型数据

a. 控制器

在 HTML 元素中通过 AngularJS 指令 ng-controller 指定控制器。

<div ng-app="myApp" ng-controller="HelloController">
    <p>Hello, {{text}}</p>
</div>

通过以下 JavaScript 代码创建控制器管理模型数据。

app.controller("HelloController",function($scope){
    $scope.text = 'AngularJS';
});
b. 模型数据

通过 AngularJS 的指令 ng-model 动态设定数据内容或通过 $scope 对象绑定数据内容。

<div ng-app="myApp" ng-controller="HelloController">
    <input type="text" ng-model="text">
    <p>Hello, {{text}}</p>
</div>

表达式

1. AngularJS 表达式

表达式的作用在于“输出”数据，将数据绑定到 HTML 元素中。

AngularJS 的表达式编写在双花括号中：{{ expression }}，与 JavaScript 表达式很相似，可以包含文字、运算符和变量等。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>表达式</title>
    <script src="js/angular.js"></script>
</head>
<body ng-app>
    <p>第一个表达式:{{ 5 + 5 }}</p>
</body>
</html>

2. AngularJS 数字

AngularJS 中的数字用法与 JavaScript 中的数字用法很相似，相应运算符的用法也是很类似的。

<div ng-app ng-init="quantity=1;cost=5">
    <p>总价： {{ quantity * cost }}</p>
</div>

3. AngularJS 字符串

AngularJS 中的字符串用法与 JavaScript 中的字符串用法很相似，相应运算符的用法也是很类似的。

<div ng-app ng-init="firstName='John';lastName='Doe'">
    <p>姓名： <span ng-bind="firstName + ' ' + lastName"></span></p>
</div>

4. AngularJS 对象

AngularJS 中的对象用法与 JavaScript 中的对象（实际也可以叫做 JSON 格式）用法很相似。

<div ng-app ng-init="person={firstName:'John',lastName:'Doe'}">
    <p>姓为 <span ng-bind="person.lastName"></span></p>
</div>

5. AngularJS 数组

AngularJS 中的数组用法与 JavaScript 中的数组用法很相似。

<div ng-app="" ng-init="points=[1,15,19,2,40]">
    <p>第三个值为 {{ points[2] }}</p>
</div>

关于运算符

JavaScript 中的常用运算符在 AngularJS 都可以使用。

6. AngularJS 与 JavaScript 表达式的不同

  • AngularJS 表达式编写在 HTML 中。
  • AngularJS 表达式不支持条件判断、循环及异常。
  • AngularJS 表达式支持 过滤器。

指令

AngularJS 的指令是通过 HTML 元素的属性来扩展 HTML。内置指令的写法带有前缀 ng-。

1. ngApp 指令

ng-app指令用于将当前 HTML 元素作为模块的根元素（所有的 AngularJS 应用都必须要有一个跟元素）。

<div ng-app="myApp"></div>
<script>
    var app = angular.module('myApp',[]);
</script>

值得注意的是：

  • HTML 页面中只允许一个 ng-app 指令。如果存在多个 ng-app 指令，则只有第一个有效。
  • 所有的 HTML 元素都支持 ng-app 指令。

2. ngInit 指令

ng-init 指令用于进行应用的数据初始化，该指令的值使用的是 AngularJS 表达式。

<div ng-app="" ng-init="myText='Hello World!'">
    <h1>{{myText}}</h1>
</div>

值得注意的是： ng-init 指令尽量少使用，建议在控制器（ng-controller）中执行。

3. ngBind 指令

ng-bind 指令用于绑定 HTML 元素到应用的数据。该指令可以替代表达式的用法。

  • 表达式用法
<div ng-app="" ng-init="myText='Hello World!'">
    <h1>{{myText}}</h1>
</div>
  • ng-bind 指令用法
<div ng-app="" ng-init="myText='Hello World!'">
    <h1 ng-bind="myText"></h1>
</div>

4. ngRepeat 指令

ng-repeat 指令用于循环输出指定数据到 HTML 元素中。

ng-repeat 指令使用的集合必须是数组或对象。

  • 使用数组循环输出表格：
<table ng-controller="myCtrl" class="table table-bordered">
    <tr ng-repeat="x in records">
        <td>{{x.Name}}</td>
        <td>{{x.Country}}</td>
    </tr>
</table>

<script>
var app = angular.module("myApp", []);
app.controller("myCtrl", function($scope) {
    $scope.records = [
       {
            "Name" : "Alfreds Futterkiste",
            "Country" : "Germany"
        },{
            "Name" : "Berglunds snabbköp",
            "Country" : "Sweden"
        },{
            "Name" : "Centro comercial Moctezuma",
            "Country" : "Mexico"
        },{
            "Name" : "Ernst Handel",
            "Country" : "Austria"
        }
    ]
});
</script>
  • 使用对象循环输出表格：
<table ng-controller="myCtrl" class="table table-bordered">
    <tr ng-repeat="(x, y) in myObj">
        <td>{{x}}</td>
        <td>{{y}}</td> 
    </tr>
</table>

<script>
var app = angular.module("myApp", []);
app.controller("myCtrl", function($scope) {
    $scope.myObj = {
        "Name" : "Alfreds Futterkiste",
        "Country" : "Germany",
        "City" : "Berlin"
    }
});
</script>

5. ngIf 指令

ng-if 指令用于当表达式的结果为 false 时移除 HTML 元素。

如果表达式的结果为 true 时，会添加被移除的元素，并显示在 HTML 页面中。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script src="js/angular.js"></script>
</head>
<body ng-app>
    保留 HTML: <input type="checkbox" ng-model="myVar" ng-init="myVar = true">

    <div ng-if="myVar">
        <h1>Welcome</h1>
        <p>Welcome to my home.</p>
        <hr>
    </div>
</body>
</html>

MVC模型

Model（模型）

Model 即模型数据，在 AngularJS 中用于存储指定作用域范围的变量值。

AngularJS 中的模型数据可以通过以下三种方式进行操作：

  • 通过使用 ng-init 指令初始化模型数据。（这种方式中Model和View混合在一起，不能清晰划分。不推荐使用）
<div ng-app="" ng-init="myText='Hello World!'">
    <h1>{{myText}}</h1>
</div>
  • 通过 Controller（控制器）对模型数据进行操作。（有效地将Model与View进行分离）
<div ng-app="myApp" ng-controller="HelloController">
    <p>Hello, {{text}}</p>
</div>
<script>
    var app = angular.module("myApp",[]);
    app.controller("HelloController",function($scope){
        $scope.text = 'AngularJS';
    });
</script>
  • 通过使用 ng-model 指令由用户动态操作模型数据。（这种方式是实现双向数据绑定的）
<body ng-app>
    <input type="text" ng-model="data.cls">
    <div class="{{data.cls}}">这是一个{{data.cls}}</div>
</body>

View（视图）

View 即视图，在 AngularJS 中就是模型数据在 HTML 模板中的运行效果。

AngularJS 中的视图，主要是依靠客户端模板（HTML模板）实现。

客户端模板 = HTML标签 + 指令 + 表达式

关于指令和表达式可参考具体内容。

视图的作用具有如下两种：

  • 将模型数据通过客户端模板展示给用户。
  • 接收用户输入的数据并传递给Model（模型）。（这种主要依靠表单完成）

Controller（控制器）

Controller 即控制器，负责创建、修改和删除模型数据。具体语法如下：

controller(name, constructor)
参数 类型 描述
name String或Object 控制器名称，或者控制器对象的映射（键是控制器名称，值为处理的回调函数）
constructor Function 控制器的构造函数

MVC模型构建数据

1）MVC模型构建简单数据

<!DOCTYPE html>
<html>
<head lang="en">
    <meta charset="UTF-8">
    <link rel="stylesheet" href="css/bootstrap.css"/>
    <script src="js/angular.js"></script>
    <title>MVC模型构建简单数据</title>
</head>
<body ng-app="myApp">
    <div class="container"  ng-controller="c1">
        <h3>使用 MVC模式创建数据并呈现数据</h3>
        <span>{{name + "," + age}}</span>
    </div>
    <div class="container"  ng-controller="c2">
        <h3>使用 MVC模式创建数据并呈现数据</h3>
        <span>{{'c2的数据：'+name}}</span>
        <br/>
        <span>{{'c2的数据：'+age}}</span>
    </div>
    <script>
        angular.module('myApp',['ng'])
        .controller('c1',function($scope) {
            $scope.name = "jerry";
            $scope.age = 34;
        }).controller('c2',function($scope){
            $scope.name = "mary";
            $scope.age = 20;
        })
    </script>
</body>
</html>

2）MVC模型构建复杂数据

<!DOCTYPE html>
<html>
<head lang="en">
    <meta charset="UTF-8">
    <link rel="stylesheet" href="css/bootstrap.css"/>
    <script src="js/angular.js"></script>
    <title>MVC模型构建复杂数据</title>
</head>
<body ng-app="myApp">
    <h2>model的类型</h2>
    <div class="container bg-primary" ng-controller="c1">
        <h3 class="page-header">显示数组数据</h3>
        <ul>
            <li ng-repeat="s in scores">{{s}}</li>
        </ul>
        <ul>
            <li ng-repeat="(k,v) in scores">{{k}} -- {{v}}</li>
        </ul>
    </div>
    <div class="container bg-info" ng-controller="c2">
        <h3 class="page-header">显示对象数据</h3>
        姓名：{{stu.name}},年龄：{{stu.age}},分数：{{stu.score}}<br>
        姓名：<span ng-bind="stu.name"></span>，年龄：<span ng-bind="stu.age"></span>，分数：<span ng-bind="stu.score"></span>
    </div>
    <div class="container bg-info" ng-controller="c3">
        <h3 class="page-header">显示对象数组</h3>
        <table class="table table-bordered">
            <thead>
                <tr>
                    <th>序号</th>
                    <th>姓名</th>
                    <th>年龄</th>
                    <th>分数</th>
                </tr>
            </thead>
            <tbody>
                <tr ng-repeat="(k,v) in stuList">
                    <td ng-bind="k+1"></td>
                    <td ng-bind="v.name"></td>
                    <td ng-bind="v.age"></td>
                    <td ng-bind="v.score"></td>
                </tr>
            </tbody>
        </table>
    </div>
    <script>
        angular.module('myApp',['ng']).controller("c1",function($scope){
            $scope.scores = [56,90,100,43,21];
        }).controller("c2",function($scope){
            $scope.stu = new Object();
            $scope.stu.name = "jerry";
            $scope.stu.age = 32;
            $scope.stu.score = 98;
        }).controller("c3",function($scope){
            $scope.stuList = [
                {name:"mary",age:23,score:45.56},
                {name:"jerry",age:18,score:90},
                {name:"tom",age:21,score:76}];
            $scope.stuList.push( {name:"heny",age:24,score:89});
        })
    </script>
</body>
</html>

其他指令

显示指令

1. ngSrc指令

ng-src指令是 AngularJS 用于替代 src 属性。

<div ng-init="myVar='angular.jpg'">
    <h1>Angular</h1>
    <img ng-src="{{myVar}}">
</div>

2. ngShow和ngHide指令

a）ngShow指令

ng-show指令在表达式为 true 时，显示指定的 HTML 元素。否则隐藏指定的 HTML 元素。

显示 HTML: <input type="checkbox" ng-model="myVar">
<div ng-show="myVar">
    <h1>Welcome</h1>
    <p>Welcome to my home.</p>
</div>
b）ngHide指令

ng-hide指令在表达式为 true 时，隐藏指定的 HTML 元素。ng-hide指令为 AngularJS 的预定义类，设置 HTML 元素的 display 为 none。

隐藏 HTML: <input type="checkbox" ng-model="myVar">
<div ng-hide="myVar">
    <h1>Welcome</h1>
    <p>Welcome to my home.</p>
</div>

3. ngChecked指令

ng-checked指令用于设置复选框或单选按钮的 checked 属性。如果 ng-checked 属性值返回 true，复选框或单选按钮将会被选中。

<body ng-app="">
    <p>My:</p>
    <input type="checkbox" ng-model="all"> Check all<br><br>

    <input type="checkbox" ng-checked="all">Volvo<br>
    <input type="checkbox" ng-checked="all">Ford<br>
    <input type="checkbox" ng-checked="all">Mercedes
</body>

4. ngDisabled指令

ng-disabled指令用于设置表单输入字段的 disabled 属性值。如果 ng-disabled 中的表达式返回 true，则表单字段将被禁用。

禁用表单输入域: <input type="checkbox" ng-model="all">
<br>
<input type="text" ng-disabled="all">
<input type="radio" ng-disabled="all">
<select ng-disabled="all">
    <option>Female</option>
    <option>Male</option> 
</select>

5. ngStyle和ngClass指令

a）ngStyle指令

ng-style指令为 HTML 元素添加 style 属性。

值得注意的是：ng-style指令的值必须是对象，表达式返回的也必须是对象。

<body ng-app="myApp" ng-controller="myCtrl">
    <h1 ng-style="myObj">达内培训</h1>
    <script>
    var app = angular.module("myApp", []);
    app.controller("myCtrl", function($scope) {
        $scope.myObj = {
            "color" : "white",
            "background-color" : "coral",
            "font-size" : "60px",
            "padding" : "50px"
        }
    });
    </script>
</body>
b）ngClass指令

ng-class指令用于给 HTML 元素动态绑定一个或多个 CSS 类。

ng-class指令的值可以是字符串、对象，或一个数组。

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <script src="angular.min.js"></script>
    <style>
        .sky {
            color:white;
            background-color:lightblue;
            padding:20px;
            font-family:"Courier New";
        }
        .tomato {
            background-color:coral;
            padding:40px;
            font-family:Verdana;
        }
    </style>
</head>
<body ng-app="">

    <p>选择一个类:</p>
    
    <select ng-model="home">
        <option value="sky">天空色</option>
        <option value="tomato">番茄色</option>
    </select>
    
    <div ng-class="home">
      <h1>Welcome Home!</h1>
      <p>I like it!</p>
    </div>

</body>
</html>

事件指令

ngClick指令

ng-click指令用于替代 click 事件。

<body ng-app>
    <p>点击按钮:</p>
    <button ng-click="count = count + 1" ng-init="count = 0">OK</button>
    <p>按钮被点击了 {{count}} 次。</p>
</body>

ngBlur指令

ng-blur指令用于告知 HTML 元素在失去焦点时需要执行的表达式。

值得注意的是：ng-clur指令不会覆盖原生的 onblur 事件。如果触发该事件，ng-blur与原生的 onblur 事件都会执行。

<body ng-app>
    <input ng-blur="count = count + 1" ng-init="count=0" />
    <h1>{{count}}</h1>
</body>

ngFocus指令

ng-focus指令用于告知 HTML 元素在获取焦点时需要执行的表达式。

值得注意的是：ng-focus指令不会覆盖原生的 onfocus 事件。如果触发该事件，ng-focus与原生的 onfocus 事件都会执行。

<body ng-app>
    <input ng-focus="count = count + 1" ng-init="count=0" />
    <h1>{{count}}</h1>
</body>

ngDblclick指令

ng-dblclick指令用于在鼠标双击 HTML 元素时需要执行的操作。

值得注意的是：ng-dblclick指令不会覆盖元素的原生 ondblclick 事件。鼠标双击时，ng-dblclick指令与原生的 ondblclick 事件将都会执行。

<body ng-app>
    <p>鼠标双击标题:</p>
    <h1 ng-dblclick="count = count + 1" ng-init="count=0">Welcome</h1>
    <p>双击了 {{count}} 次。</p>
<body ng-app>
